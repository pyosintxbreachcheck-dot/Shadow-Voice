# 🎙️ Shadow Voice

> Secure • Private • Peer-to-Peer Voice Communication

Shadow Voice is a lightweight browser-based voice calling platform built with **WebRTC** and **PeerJS**. It enables users to create temporary voice rooms and communicate directly without requiring accounts, phone numbers, or software installation.

---

## ✨ Features

* 🎤 Real-time voice communication
* 🔒 End-to-end encrypted WebRTC audio streams
* ⚡ Peer-to-peer connectivity
* 🚀 No registration required
* 🏠 Instant room creation
* 🔗 Shareable room codes & invite links
* 📱 Mobile & desktop compatible
* 🎭 Nickname-based participation
* 🎚️ One-click microphone mute/unmute
* 🌐 Runs entirely in the browser

---

## 🛡️ Privacy

Shadow Voice is designed with privacy in mind.

* No account creation
* No phone number verification
* No message storage
* No audio recording by the application
* Direct peer-to-peer audio transmission
* Encrypted WebRTC media channels

**Note:** Privacy depends on participant device security and proper room code handling.

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/shadow-voice.git
cd shadow-voice
```

### 2. Open Application

Simply open:

```bash
call.html
```

or serve using:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

---

## 📞 How It Works

### Create Room

1. Enter your callsign.
2. Click **Create Room**.
3. Allow microphone access.
4. Share room code.

### Join Room

1. Enter your callsign.
2. Click **Join Room**.
3. Enter room code.
4. Allow microphone access.
5. Start talking instantly.

---

## 🔐 Security Model

Shadow Voice uses:

* WebRTC encrypted media transport
* PeerJS signaling
* Direct peer-to-peer connections
* Temporary room sessions

The application does not maintain a user database or store voice data.
